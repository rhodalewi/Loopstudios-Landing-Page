# Loopstudios Landing Page (HTML & Sass)

This project is an implementation of the Loopstudios Landing Page challenge from Frontend Mentor, built using semantic HTML and Sass. The goal was to strengthen my understanding of core CSS concepts by recreating the same interface without relying on utility frameworks.

## Overview

In this challenge, I focused on writing structured, maintainable styles using Sass while maintaining a pixel-accurate and responsive layout.

## Key Features

* Fully responsive layout across mobile and desktop
* Mobile navigation menu with smooth slide-in animation
* Custom hover effects using CSS pseudo-elements
* Modular and organized Sass structure

## Tech Stack

* HTML5 (semantic markup)
* Sass (modular and reusable styling)
* CSS3
* JavaScript

## Highlights

* Structured styles using Sass partials for better scalability and readability
* Implemented animations and transitions manually without utility classes
* Recreated interactive UI elements (e.g., navigation underline effects) using pseudo-elements
* Focused on clean separation of concerns between structure and styling
